# PropertyIQ Valuation Methodology

## Valuation Components

### Comparable Sales Method
Weight: 40%

### Rental Yield Method
Weight: 20%

### Replacement Cost Method
Weight: 25%

### Affordability Method
Weight: 15%

## Weighted Fair Value Formula

Fair Value =
(
Comparable Sales × 0.40
+
Rental Yield × 0.20
+
Replacement Cost × 0.25
+
Affordability × 0.15
)
